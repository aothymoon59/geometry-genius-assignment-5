What is event bubble?

when a user clicks on a button on the web page, the event handler under that button propagates the event to its parent, its parent, and its parent again to the root element Html. The event notifies all elements above it that someone has clicked on it.

What is a callback function and why will you use it?

A callback function is when a function is passed as an argument to a function, it is executed after the main function is executed. So it is called callback function

We can use callback function in 2 ways.

1. as Anonymous Function

2. as Regular Function

We need callback functions to make sure that a function is only going to get executed after a specific task is completed and not before its completion.

in the case of an event listener, we first listen for a specific event, and if the function detects it, then only the callback function is executed.
